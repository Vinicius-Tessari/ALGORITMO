# Exercícios de Funções (Nível Intermediário)

---

## 1️ Função com validação

Crie uma função que receba um número.

* Se for negativo → retorne "Número inválido"
* Se for positivo → retorne a raiz quadrada

### Exemplo:

```python
print(raiz(9))
```

Saída esperada:

```
3.0
```

---

## 2️ Contador de vogais

Crie uma função que receba uma string e retorne **quantas vogais existem nela**.

### Exemplo:

```python
print(contar_vogais("programacao"))
```

Saída esperada:

```
5
```

---

## 3️ Função com lista

Crie uma função que receba uma lista de números e retorne:

* a soma total
* a média

### Exemplo:

```python
print(calcular([2, 4, 6]))
```

Saída esperada:

```
(12, 4.0)
```

---

## 4️ FizzBuzz com função

Crie uma função que receba um número e:

* Se for múltiplo de 3 → retorne "Fizz"
* Se for múltiplo de 5 → retorne "Buzz"
* Se for dos dois → retorne "FizzBuzz"
* Caso contrário → retorne o número

### Exemplo:

```python
print(fizzbuzz(15))
```

Saída esperada:

```
FizzBuzz
```

---

## 5️ Saudação por horário

Crie uma função que receba um horário no formato "HH:MM" e retorne:

* "Bom dia" (05:00–11:59)
* "Boa tarde" (12:00–17:59)
* "Boa noite" (18:00–04:59)

### Exemplo:

```python
print(saudacao_horario("14:30"))
```

Saída esperada:

```
Boa tarde
```

Esses exercícios trabalham:

* validação de dados
* uso de condições (`if`, `elif`, `else`)
* manipulação de strings
* listas
* retorno de múltiplos valores
